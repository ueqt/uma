Components

UmaTimerPicker

This is the main class of the component.

Inputs

Name Description
hideButtons: boolean Hide the buttons (confirm/cancel)
umaTimerPickerFormat: UmaTimerPickerFormat Define the format that timer-picker will output the selected time (default: 12)
umaTimerPickerMin: string Define the minimum allowed time (default: 00:00 am, accepts both formats)
umaTimerPickerMax: string Define the maximum allowed time (default: 12:00 pm, accepts both formats)
btnCancel: string Cancel button text (default: Cancel)
btnConfirm: string Confirm button text (default: Confirm)

Outputs

Name Description
selected: string Output the selected timer

Directives

UmaTimerPickerOriginDirective

This directive is used on the trigger component. If we associate the directive an input text, when the input is focused the timer picker will open on the screen

UmaConnectedTimerPickerDirective

Directive used on the UmaTimerPickerComponent to connect to an element ( UmaTimerPickerOriginDirective)

Inputs

Name Description
umaConnectedTimerPickerOrigin: UmaTimerPickerOriginDirective Element that timer picker has to connect
