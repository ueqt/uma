Overview

UmaColorPicker

When import the UmaColorPickerModule, you can provide a configuration for the empy value.

 import {{ '{' }} UmaColorPickerModule {{ '}' }} from 'uma';

  @NgModule({{ '{' }}
    ...
    imports: [
      UmaColorPickerModule.forRoot({{ '{' }}
        empty_color: 'transparent',
        used_colors: ['#000000', '#FFF555']
      {{ '}' }})
    ]
    ...
  {{ '}' }})

The default value is none, but you can set anything you want.

For the @Input and @Output of the component, check API tab.
